# 🚀 HWatch Website

## 🔍 Overview
A website that aims to display watches in a beautiful, modern style using various types of animation.

---

## 🛠️ Technologies Used


- **React.js** as the React framework
- **GSAP** for Motion
- **framer-motion** for Motion 
- **react-dom** for icons
---

## ✨ Features

-⌚ Explore a premium collection of watches with a clean UI
-🔍 Detailed product pages with high-quality visuals and info
- 🚀 Performance optimized (CLS, bundle size, code splitting)
---

### To Install and Run the Project
1. Clone the repository:
    ```bash
    https://github.com/Lotfy12/HWatch.git
    ```
2. Navigate to the project directory:
    ```bash
    cd project-directory
    ```
3. Install the dependencies:
    ```bash
    npm install
    # or
    yarn install
    ```
4. Run the development server:
    ```bash
    npm run dev
    # or
    yarn dev
    ```
